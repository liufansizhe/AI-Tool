## 平台特点

### 小红书
- 图文为主，封面图很重要
- 标题党有效，多用emoji
- 标签带满10个
- 评论区互动要积极

### 抖音
- 视频前3秒决定完播率
- 口播文案要口语化
- BGM选择热门音乐
- 引导关注和评论
